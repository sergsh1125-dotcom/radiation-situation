# Radiation Mapping Tool ☢️

Програма для моніторингу радіації з візуалізацією на карті та генерацією PDF-звітів.

### Функції:
- Введення координат замірів через GUI.
- Побудова ізоліній (0.03, 0.3, 1.0, 5.0 мЗв/год).
- Відображення прапорців з підписами значень прямо на карті.
- Автоматичне збереження бази даних у CSV.
- Створення PDF-звітів з підтримкою української мови.

### Запуск:
1. Встановіть бібліотеки: `pip install -r requirements.txt`
2. Додайте файл `DejaVuSans.ttf` для коректного PDF.
3. Запустіть: `python main.py`
